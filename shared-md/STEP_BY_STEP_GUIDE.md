#### **Bước 1.1: Kiểm tra hệ thống**

```bash

node --version    # Cần >= 16.0.0
npm --version     # Cần >= 8.0.0

# Kiểm tra Python
python3 --version # Cần >= 3.8
pip3 --version    # Cần có pip3

# Kiểm tra Git
```

#### **Bước 1.2: Cài đặt thiếu (nếu cần)**

## 🆘 **TROUBLESHOOTING**

### **Lỗi thường gặp:**

#### **1. Port đã được sử dụng**

```bash
# Tìm process sử dụng port
lsof -i :8080
lsof -i :3001
lsof -i :8000

# Kill process
kill -9 <PID>
```

#### **2. Dependencies lỗi**

```bash
# Reinstall dependencies
rm -rf node_modules package-lock.json
npm install --legacy-peer-deps

# Python dependencies
cd ai-service
pip3 install -r requirements.txt --force-reinstall
```

#### **3. Services không start**

```bash
# Restart all services
./deploy.sh stop
./deploy.sh start

# Hoặc restart manual
pkill -f "node|python|serve"
./quick-setup.sh deploy
```
